# Rag-agent-system

## Setup Instructions

Follow these steps to set up and run the project.

### 1. Create and Activate Virtual Environment
#### On macOS/Linux:
```sh
python -m venv venv
source venv/bin/activate
```

#### On Windows:
```sh
python -m venv venv
venv\Scripts\activate
```

### 2. Install Dependencies
```sh
pip install -r requirements.txt
```
### 3. Run the Project
```sh
python app/main.py
```
